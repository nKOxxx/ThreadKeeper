# GitHub Launch Setup

## 🚀 Quick Setup (Do These Now)

### 1. Enable GitHub Discussions
- Go to: https://github.com/nKOxxx/ThreadKeeper/settings
- Under "Features" → Check "Discussions"
- Save

### 2. Create GitHub Discussions Categories
Once discussions are enabled, create these categories:

**Announcements**
- For: Major updates, releases, announcements
- Private: No (public)

**General**
- For: General discussion about Threadkeeper
- Private: No (public)

**Ideas & Feature Requests**
- For: Feature requests and ideas for v0.2+
- Private: No (public)

**Q&A**
- For: Questions about installation, usage, troubleshooting
- Private: No (public)

**Show & Tell**
- For: Share how you're using Threadkeeper, use cases, stories
- Private: No (public)

### 3. Create Initial Discussions Post

Go to https://github.com/nKOxxx/ThreadKeeper/discussions/new

**Category**: Announcements
**Title**: "Threadkeeper v0.1.0 is live! 🧵"
**Body**:
```
Welcome to Threadkeeper! 

This is the community hub for discussion about automatic context persistence for Claude Code.

## What is Threadkeeper?
Never lose context between Claude Code sessions again. One command to install, zero configuration, automatic injection of your previous decisions and technologies into new sessions.

## Quick Links
- 📖 [README](https://github.com/nKOxxx/ThreadKeeper#readme) - Overview and features
- 🚀 [Quick Start](https://github.com/nKOxxx/ThreadKeeper/blob/main/QUICKSTART.md) - Get started in 2 minutes  
- 📚 [User Guide](https://github.com/nKOxxx/ThreadKeeper/blob/main/USER_GUIDE.md) - Complete documentation
- 🔒 [Security](https://github.com/nKOxxx/ThreadKeeper/blob/main/SECURITY.md) - Security features and threat model

## Installation
```bash
npx threadkeeper install
```

## 🚨 Common Issues to Address Early

**Issue: "It didn't work on first install"**
→ Response: Point to USER_GUIDE.md troubleshooting section

**Issue: "Will my data leak?"**
→ Response: Link to SECURITY.md, explain it's all local

**Issue: "How does this compare to Claude-Mem?"**
→ Response: Link to COMPARISON.md

**Issue: "When will v0.2 be released?"**
→ Response: "Working on semantic search + encryption, ETA TBD"
